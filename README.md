# Bean There Coffee Shop

A modern, responsive landing page for Bean There coffee shop built with Next.js 14 and Tailwind CSS.

## Features

- Hero section with brand tagline: "Life's too short for bad coffee"
- Menu showcase with pricing (Espresso, Latte, Cold Brew)
- About section highlighting the coffee shop's story
- Contact footer with address: 123 Brew St, Portland OR
- Warm brown and cream color scheme
- Fully responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Live Demo

Visit the live site: https://bean-there-cafe.vercel.app

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
